## Description
VN-Backtest is a C-based stock trading simulator that backtests three technical analysis strategies on Vietnamese stock market data (VN30, VNMidcap, and full market indices). The application processes historical daily price data, calculates technical indicators (MACD, Simple Moving Averages, Bollinger Bands), and executes automated trading strategies: MACD Histogram Reversal, SMA Crossover with Volume Spike, and Bollinger Bands Breakout. It provides comprehensive trade logs and performance metrics including profit percentages and portfolio values to help evaluate trading strategy effectiveness on historical data.
